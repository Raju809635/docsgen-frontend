# docsgen-frontend (Vercel)

React (Vite) + Tailwind + Mermaid UI for the AI Documentation Generator.

## Run locally

```powershell
npm install
copy .env.example .env
npm run dev
```

## Deploy to Vercel

- Set env var `VITE_API_URL` to your Render backend URL (for example: `https://docsgen-backend.onrender.com`).
- Build output: `dist`
